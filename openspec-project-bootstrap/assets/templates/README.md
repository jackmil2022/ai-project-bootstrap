# Templates

These files are intended to be copied into a target repository and then adapted:

- `AGENTS.template.md`
  Root-level long-lived collaboration rules.

- `PACKAGE_AGENTS.template.md`
  Directory-scoped rules for package, domain, or module folders.

- `openspec-config.yaml.example`
  Example `openspec/config.yaml` content with project context and per-artifact rules.

- `CODEX_MANUAL.template.md`
  Optional heavier-weight project manual for teams that want stronger governance.

- `review.md.example`
  Example review artifact for projects that want an explicit review file per change.
