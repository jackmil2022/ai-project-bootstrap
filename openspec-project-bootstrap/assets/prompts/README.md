# Prompts

Use these prompt files as reusable phase-specific instructions:

- `official-init.md`
  Initialize OpenSpec with the official structure.

- `augment-project-rules.md`
  Add project governance on top of an existing official baseline.

- `generate-package-agents.md`
  Generate package or module scoped `AGENTS.md` files.

- `explore-requirement.md`
  Explore a requirement before creating change artifacts.

- `create-change.md`
  Create proposal, specs, design, and tasks through official OpenSpec.

- `execute-apply.md`
  Implement the change with TDD and verification expectations.

- `review-after-apply.md`
  Run the project-level review pass after implementation.

- `archive-change.md`
  Archive the change after review and user confirmation.
